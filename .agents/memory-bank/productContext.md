# Product Context: IMEC 3D Exploratory Map

## Core Objective
Build a highly interactive, free-exploration map visualizing the India-Middle East-Europe Economic Corridor (IMEC).

## Aesthetic & Design Philosophy
- **Light Academic / Enterprise Minimalist**: Crisp, professional, and accessible.
- **Editorial Tone**: Looks like a high-end intelligence brief or premium newspaper graphic (e.g., NYT, WSJ, FT).
- **Typography**: Clean, sans-serif fonts governing hierarchy.
- **Whitespace**: Generous padding and margins to prevent cognitive overload.

## Data Vectors
- Infrastructural anchoring over three domain layers: 
  1. IMEC Railway
  2. Blue-Raman Subsea Cables
  3. Strategic Data Centers

## Target Experience
- An elegant spatial dashboard where interactions are subtle, colorblind-accessible, and performant.
- Users sequentially filter architectural layers and spatially navigate via intuitive Sidebar controls mapping to critical sub-regions (India, Middle East, Europe).
